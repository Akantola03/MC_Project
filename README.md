# MC_Project
Mobile Computing project:

My project is a simulated messaging app, where every user message triggers a response. User can also delete individual messages by pressing them long.
When notifications are enabled from profilepage, the "bots" replies trigger a notification, if the app is running in background.


Features:

Custom made splashscreen.

User can send messages to chat.

Individual messages can be deleted by longpress.

Every message sent by the user triggers a precoded response (4 options in total).

Messages are stored in a room database.

Notifications can be enabled from the profilescreen. 

If notifications are enabled, and the user leaves the app running in background before ChatBOT responds, a new notification is triggered. 
Clicking this notification takes the user straigth to the chat.
